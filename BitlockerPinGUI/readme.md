# BitlockerPinGUI

A simple PowerShell GUI tool to set or change the BitLocker PIN on Windows devices.


## Requirements

- Windows 10/11
- PowerShell 5.1 or later
- BitLocker enabled

## Disclaimer

Use at your own risk. Test in a safe environment before deploying widely.
